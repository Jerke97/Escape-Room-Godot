# Escape Room (Godot)
Um pequeno jogo de Adventure Point-and-Click desenvolvido na engine Godot 4.
Com um loop de gameplay curto, seu objetivo é encontrar uma saída.

## Objetivo do Desenvolvedor
Meu objetivo com o desenvolvimento deste jogo foi aprender sobre Godot Engine, GDScript, Pixel-Art e Fundamentos de Point-and-Click, na prática, tendo que lidar com os desafios de criar um jogo. 

## Gameplay
A jogabilidade é muito simples, como qualquer Point-and-Click, basta clicar para interagir e coletar.

## Tecnologias Usadas
- **Engine:** Godot 4.3 (GDScript)
- **Arte:** Pixel Art (Photopea e Pixel Art Village)
- **Conceitos:** Máquina de Estados, Autoload (Singleton), Sinais e Tweens.

## Como rodar o projeto
1. Baixe o código fonte.
2. Importe o arquivo `project.godot` na Godot Engine.
3. Dê Play!
